Description du projet

FormLab est une application web qui permet de remplir automatiquement des formulaires PDF à partir des informations saisies par les utilisateurs. Développée dans le cadre du BTS SIO, c'est mon premier projet complet en PHP/MySQL.

Fonctionnalités principales

  • ✓ Gestion des utilisateurs : Création de compte, connexion, modification des informations
  • ✓ Création de formulaires en 3 étapes avec validation en temps réel
  • ✓ Génération automatique de PDF avec les données saisies
  • ✓ CRUD complet : Liste, modification, suppression des formulaires
  • ✓ Gestion des sessions avec PHP (POST/GET)

Étapes du formulaire

Étape 1 : Informations générales (titre, périodes, adresse)
Étape 2 : Détails du contrat (partenaires, activités, lieu)
Étape 3 : Clauses (contributions, bénéfices, juridiction)
